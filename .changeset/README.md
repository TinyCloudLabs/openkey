# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the changesets repository](https://github.com/changesets/changesets).

## Publishing Packages

This repo publishes the following packages to npm:
- `@openkey/sdk` - The main OpenKey SDK for browser applications
- `@openkey/types` - Shared TypeScript types

### Adding a changeset

```bash
bun changeset add
```

### Versioning packages

```bash
bun changeset version
```

### Publishing to npm

```bash
bun run publish
```

Or let the CI workflow handle it automatically on merge to main.
